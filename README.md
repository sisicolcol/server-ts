# server

before start
```
npm install
1. .env.example이 있는 경로에 .env 파일을 생성합니다
2. 노션 워크스페이스 백엔드 - .env 파일로 들어갑니다
3. 노션에 있는 내용을 .env 파일에 복붙합니다
4. npm run start
```

how to start
```
npm run start
```

default page
```
http://localhost:3000
```

generating api on swagger
```
node .\swagger-generator.ts
```

api page
```
http://localhost:3000/swagger/
```
